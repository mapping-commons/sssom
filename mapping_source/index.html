    mapping_source

    URI: sssom:mapping_source

    Applicable to: Mapping

    Description

    The mapping set this mapping was originally defined in. mapping_source is used for example when merging multiple mapping sets or deriving one mapping set from another.

    Properties

    • Range: EntityReference. The range of the element is the type of the value that can be assigned to it.

    Examples

    Example: MONDO_MAPPINGS:mondo_exactmatch_ncit.sssom.tsv

    Example value:

    MONDO_MAPPINGS:mondo_exactmatch_ncit.sssom.tsv

    Description: A reference to the mapping set that originally contained this mapping.
