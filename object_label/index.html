    object_label

    URI: sssom:object_label

    Applicable to: Mapping

    Description

    The label of object of the mapping

    Properties

    • Range: String. The range of the element is the type of the value that can be assigned to it.

    • Required?: Recommended (We suggest adding this element to the mapping or mapping set, but it is not required.

    Examples

    Example: Thickened ears

    Example value:

    Thickened ears
