    Class: MappingSetReference

    A reference to a mapping set. It allows to augment mapping set metadata from the perspective of the registry, for example, providing confidence, or a local filename or a grouping.

    URI: sssom:MappingSetReference

    classDiagram class MappingSetReference MappingSetReference : last_updated MappingSetReference : local_name MappingSetReference : mapping_set_group MappingSetReference : mapping_set_id MappingSetReference : mirror_from MappingSetReference : registry_confidence

    Slots

    Name Range Cardinality Description Info
    mapping_set_id xsd:anyURI 1..1 A globally unique identifier for the mapping set (not each individual mapping). Should be IRI, ideally resolvable. .
    mirror_from xsd:anyURI 0..1 A URL location from which to obtain a resource, such as a mapping set. .
    registry_confidence xsd:double 0..1 This value is set by the registry that indexes the mapping set. It reflects the confidence the registry has in the correctness of the mappings in the mapping set. .
    mapping_set_group xsd:string 0..1 Set by the owners of the mapping registry. A way to group . .
    last_updated xsd:date 0..1 The date this reference was last updated. .
    local_name xsd:string 0..1 The local name assigned to file that corresponds to the downloaded mapping set. .
