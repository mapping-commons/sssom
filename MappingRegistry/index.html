    Class: MappingRegistry

    A registry for managing mapping sets. It holds a set of mapping set references, and can import other registries.

    URI: sssom:MappingRegistry

    classDiagram class MappingRegistry MappingRegistry : documentation MappingRegistry : homepage MappingRegistry : imports MappingRegistry : mapping_registry_description MappingRegistry : mapping_registry_id MappingRegistry : mapping_registry_title MappingRegistry : mapping_set_references

    Slots

    Name Range Cardinality Description Info
    mapping_registry_id EntityReference 1..1 The unique identifier of a mapping registry. .
    mapping_registry_title xsd:string 0..1 The title of a mapping registry. .
    mapping_registry_description xsd:string 0..1 The description of a mapping registry. .
    imports xsd:anyURI 0..* A list of registries that should be imported into this one. .
    mapping_set_references MappingSetReference 0..* recommended A list of mapping set references. .
    documentation xsd:anyURI 0..1 A URL to the documentation of this mapping commons. .
    homepage xsd:anyURI 0..1 A URL to a homepage of this mapping commons. .
